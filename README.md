# MTG Price Tracker 📊

**Just open index.html in your browser.** Nothing to install.

- ✅ No Python, no terminal, no setup
- ✅ All data stored in your browser (localStorage)
- ✅ Backup/restore with JSON export
- ✅ Dark theme, charts, portfolio management
- ✅ Quantity tracking per product

### Features

| Tab | What you can do |
|-----|----------------|
| **Dashboard** | See all products, prices, bar chart |
| **Products** | Add products, set prices, view charts |
| **Portfolios** | Create named groups, add products with quantities |
| **Report** | Download a self-contained HTML report with charts |

### Quick Start

1. Download this repo
2. Open `index.html` in your browser
3. Click **Seed Demo** to try it
4. Or add your own products manually

### Data Stays in Your Browser

Everything is stored in localStorage — it persists between sessions on the same device. Use the 💾/📂 buttons to backup or restore your data.

### Still Want the CLI / API Version?

The Python-based tools are available in this repo for advanced users:
- `tracker.py` — CLI interface
- `web_app.py` — Flask web server
- `api.py` — Cardmarket API integration
